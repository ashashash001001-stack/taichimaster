# PageSpeed Optimization Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Fix performance bottlenecks identified in the PageSpeed analysis: migrate all image references to WebP, replace external Lucide CDN with local inline icons for 100 article pages, improve CSS loading, lazify Google Maps, and clean up repo artifacts.

**Architecture:** Two independent subsystems: (A) direct HTML edits to `index.html` for image fallbacks, social meta tags, Maps iframe lazification, and orphaned file cleanup; (B) modify `fix_duplicates.py`'s `generate_article_html()` to use a local `js/icons.js` script tag instead of `unpkg.com/lucide@latest`, then regenerate all 100 article pages via `python3 fix_duplicates.py`. No new dependencies. No new files created beyond the plan itself.

**Tech Stack:** Python 3 (for article generation), HTML/Tailwind CSS, vanilla JavaScript, Lucide inline SVG icons (already in `js/icons.js`)

## Current State (What's Broken)

### System A: index.html

- **5 `<img>` fallback src attributes** still point to non-WebP files (`.png`, `.jpg`, `.JPG`) even though `.webp` equivalents exist in `images/` — browser loads 1.55MB `class.png` instead of 157KB `class.webp` if `<picture>` fails
- **`og:image` and `twitter:image` meta tags** (lines 55, 62) reference `class.png` — social crawlers get the 1.55MB PNG
- **JSON-LD schema** `"image"` field (line 110) and `"logo"` field (line 122) reference `class.png` and `favicon.png`
- **Google Maps iframe** loads eagerly (lines 635-645) — heavy third-party embed on page load, no lazy loading
- **Orphaned large files** in repo root: `class.png` (1.55MB), `solo.jpg`, `solo2.jpg`, `class_hall.jpg`, `class_8year.JPG`, `chromewebdata_*.report.html` (failed Lighthouse artifact)

### System B: Article Pages (generated by fix_duplicates.py)

- **100 article pages** use `<script src="https://unpkg.com/lucide@latest"></script>` — external, render-blocking JS that `index.html` has already replaced with local `js/icons.js`
- **Article pages** load `../css/tailwind.css` via `<link rel="stylesheet">` without `preload` trick or critical CSS inline
- **Article generator** also emits unused preconnects (`cdn.tailwindcss.com`, `unpkg.com`) that `index.html` doesn't have

### Impact Summary

| Issue | Pages Affected | Estimated Savings |
|-------|---------------|-------------------|
| Non-WebP fallbacks | index.html | ~2MB fewer bytes on cold cache |
| Social meta using PNG | index.html | 1.5MB per share preview |
| Google Maps eager load | index.html | ~500ms LCP |
| unpkg Lucide CDN | 100 articles | ~10KB + ~200ms per article |
| No CSS preload on articles | 100 articles | ~500ms FCP per article |

```javascript
// Click-to-load Google Maps
const mapPlaceholder = document.getElementById('map-placeholder');
const mapContainer = document.getElementById('map-container');
if (mapPlaceholder && mapContainer) {
    mapPlaceholder.addEventListener('click', () => {
        mapPlaceholder.remove();
        mapContainer.classList.remove('hidden');
        const iframe = document.createElement('iframe');
        iframe.width = '100%';
        iframe.height = '100%';
        iframe.frameBorder = '0';
        iframe.scrolling = 'no';
        iframe.marginHeight = '0';
        iframe.marginWidth = '0';
        iframe.src = 'https://maps.google.com/maps?q=香港小童群益會賽馬會油塘青少年綜合服務中心&t=&z=17&ie=UTF8&iwloc=&output=embed';
        iframe.title = 'Google Maps Location';
        iframe.className = 'absolute inset-0 w-full h-full';
        iframe.setAttribute('loading', 'lazy');
        mapContainer.appendChild(iframe);
    });
    // Add keyboard accessibility
    mapPlaceholder.setAttribute('tabindex', '0');
    mapPlaceholder.setAttribute('role', 'button');
    mapPlaceholder.setAttribute('aria-label', '點擊載入 Google 地圖');
    mapPlaceholder.addEventListener('keydown', (e) => {
        if (e.key === 'Enter' || e.key === ' ') {
            e.preventDefault();
            mapPlaceholder.click();
        }
    });
}
```
